🎮 NetHack WASM

Webブラウザで動く NetHack ゲーム実装

📚 プロジェクト概要

このプロジェクトは、NetHack という有名なダンジョン探索ゲームを Rust で完全に書き直し、WebAssembly (WASM) 経由で Web ブラウザで動作させるものです。

📖 プロジェクト構成：
Rust コア実装 → wgpu 3D レンダリング → Desktop + WASM + Unity 対応

🎯 利用可能な Examples

🌐 Interactive Browser Demo LIVE

WASM で動く NetHack を Web ブラウザで操作できる完全なデモです。 リアルタイムで ゲーム状態が更新され、複数のカメラモードで ダンジョンを探索できます。

  • Canvas 2D リアルタイムレンダリング
  • 矢印キーで プレイヤー移動
  • 5種類のカメラモード切り替え
  • ゲーム統計情報 表示
  • グリッド表示 & クロスヘア
  • 60 FPS で動作

🛠️ ローカルで実行する

GitHub Pages から直接実行するか、 ローカルマシンで HTTP サーバーを起動して実行できます。

⚠️ 注意： WASM ファイルを読み込むには HTTP サーバーが必須です。 file:// プロトコルでは動作しません。 
# Python 3 を使う場合
python3 -m http.server 8000

# Node.js を使う場合
npx http-server

# その後、ブラウザで以下を開く
http://localhost:8000

🎮 操作方法

矢印キー: プレイヤーを上下左右に移動
1-5 キー: カメラモードを切り替え
Q キー: ゲーム終了
